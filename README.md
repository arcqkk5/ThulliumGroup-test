## ThulliumGroup test project.

### About project

- Output of images by tiles
- Modal window when clicking on an image
- Deleting cards
- Search by ID
- Pagination

1. Git clone

```
git clone https://github.com/arcqkk5/ThulliumGroup-test.git
```

2. Launch

```
npm i
npm start
```
